优质软件推荐（持续更新）

发表于 2022-08-06 分类于 软件相关 阅读次数： Changyan：

本文主要介绍计算机科学与技术专业常用软件，包含作者推荐的好用软件，避免走弯路。

一、系统软件

1. 压缩软件

1.1 bandizip

bandizip是作者正在使用的软件，UI界面和解压缩速度都和优秀，支持大部分常见格式，支持预览压缩包内容，支持像压缩包添加和删除文件。

1.2 7-zip

7-zip是一款比较轻量级的解压缩软件，同样支持大部分常见格式，速度快，安装时间极短，无广告，缺点是UI界面略丑。

2. 安全软件

2.1 写在前面

  • 作者并不推荐在windows电脑上安装所谓的安全卫士等软件，这些软件会占用电脑的内存，影响电脑的使用速度。
  • 然而，如果你是小白用户，这些安全软件会有很大帮助，包括但不限于清理垃圾，修复网络，扫描病毒等等。
    下面写一些常用安全软件的注意事项。

2.2 联想电脑管家

  • 如果你使用的是联想电脑，可以尝试使用联想电脑管家，能够比较清晰的看到电脑的保修日期等信息。
  • 旧的联想电脑可以使用联想电脑管家的安装驱动功能，切记不要使用官网推荐的联想驱动管家，会出问题。
  • 使用时注意打开设置选项，将自动更新和个性化等功能关闭。
  • 此外，电脑的锁屏壁纸默认被联想电脑管家接管，可以在设置中进行关闭。

2.3 360安全卫士

  • 360并不是所谓的流氓软件，因为他提供了强大的杀毒功能和系统修复功能，高中时我教室的电脑插U盘总会中毒，只有360可以恢复被病毒隐藏的文件（其他方法较麻烦，对小白不推荐）
  • 自带的网络急救箱也十分好用。
  • 记得使用时将设置中的自动更新和个性化推荐关闭。

2.4 火绒安全

  • 火绒是近两年比较热门的轻量级杀毒软件，但实际上杀毒能力较弱，各方面不敌360
  • 但是占用内存较小，强迫症同学可安装。

2.5 写在后面

  • 正常的电脑不会有广告弹窗，而且通过安全软件的方式拦截弹窗本就是下下策。如果遇到弹窗软件，肯定能够在相关软件的设置中关闭弹窗。
  • 请大家尽量在软件对应的官网下载软件，而且安装软件的时候要注意有没有勾选捆绑安装。
  • 如果你是破解软件的爱好者，或者经常下载破解游戏或者作弊器，那么还是建议安装一个安全软件，可以帮助你及时发现捆绑软件的安装。

二、办公软件

1. 文档类

1.1 Microsoft Office

  • 简称MSOffice，是微软家的办公套件，作者正在使用的软件。如果你是windows系统，那么极其推荐使用MSOffice。
  • 常见的核心套件为Word，Excel和PowerPoint，拥有强大的功能和优异的性能。
  • 除此以外，还会用到一些其他软件，如Visio（流程图），Project（工程项目管理）等。
  • 现在购买新电脑一般会送windows家庭版激活和Office家庭版的激活，记住在windows激活的一天内一定要激活Office，不然免费的正版激活就无了。
  • 如果你想免费安装专业版的Office，可以使用Office Tools软件，能够一键部署和激活Office套件，作者正在用2019专业增强版的Office套件。

1.2 WPS Office

  • 金山文档出版的Office办公套件，对新手小白友好，
  • 有很多模板提供选择，但大部分收费。
  • 性能较MSOffice稍弱，占用内存较多。如果轻度使用的话是比较不错的选择
  • Windows、MacOS、Linux全平台通用，跨设备支持较好。

1.3 Libre Office

  • 开源的Office套件，兼容性较差，上手较前两个软件难，
  • 如果你的电脑性能低，可以尝试使用。

2. PDF类

2.1 Adobe Acrobat DC

  • Adobe公司的知名软件，支持pdf查看，批注，编辑，全能软件，建议在电脑安装一份备用。
  • 使用时注意Ctrl+K找到更新选项，将自动更新关闭。
  • 不建议设置为默认PDF查看器，因为软件体量较大，速度一般。

2.2 Microsoft Edge

没错，自带的浏览器可以完美的浏览和批注PDF，速度性能优良，推荐浏览时使用。

2.3 SumatraPDF

  • 一款非常轻量级的PDF阅读器，仅支持查看PDF，不能批注和编辑，启动速度快，
  • 在使用Sublime进行LaTeX论文写作时可作为预览文档的软件。

3. 文本编辑器类

3.1 Sublime

  • 启动速度极快，阅读代码体验极佳，
  • 缺点是对中文内容的文档显示略微奇怪，需要通过安装中文插件实现。

3.2 Notepad++

  • 尽管该软件作者有不当言论，但软件仍然十分优秀，爱国情怀慎用
  • 新手友好编辑器

3.3 EditPlus

  • 同上，新手友好编辑器

三、专业软件

1. 编程软件

1.1 DevC++

  • 轻量化C/C++编辑器，下载即用，不需要配置环境，
  • 适合程序竞赛同学使用

1.2 Visual Studio Code

  • 一般称vscode，本质上是文本编辑器
  • 插件丰富，可以通过安装插件的方式支持运行代码和项目
  • 可自定义程度很高，支持更换主题

1.3 Pycharm

  • Jetbrain家的python开发工具，业界最强

1.4 Intellij Idea

  • Jetbrain家的java开发工具，很好的支持现有的框架和技术

1.5 Eclipse

  • IBM公司的java开发工具，常用于旧框架和项目

1.6 Visual Studio

  • 微软家的开发工具，对C#支持较好
  • 面向对象设计大作业，如果选择C++语言，用VS很方便
