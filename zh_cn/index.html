    🔨 项目

    GitHub 仓库

    StaffDeck：面向企业的数字员工构建与管理平台

    面壁智能、东北大学-面壁智能数据智能联合实验室、清华大学 THUNLP 实验室、OpenBMB 与 AI9Stars

    StaffDeck 是一套面向企业的数字员工构建与管理平台，帮助专业员工将工作经验、业务流程和判断标准固化为可以持续工作的数字员工，接手重复性任务，并将个人能力沉淀为可复用、可迭代、可追溯的组织资产。

    GitHub 仓库

    PilotDeck：以工作区为中心的开源智能体操作系统

    THUNLP、ModelBest、OpenBMB 与 AI9Stars

    PilotDeck 是一款以工作区为核心的开源智能体操作系统，面向长周期的生产力场景，提供工作区隔离、白盒记忆、智能路由以及常驻后台运行等能力。

    GitHub 仓库

    UltraRAG 3.0：告别“盲盒式”开发，让每一行推理都清晰可见

    THUNLP、NEU-ModelBest数据智能联合实验室、OpenBMB 与 AI9Stars

    UltraRAG 是一个基于 MCP 的低代码 RAG 框架，将检索与生成模块标准化为 MCP Server，并提供可视化编排、评测与演示部署的一站式流程。

    📝 论文发表

    * 表示共同一作，† 表示通讯作者。

    ACL 2026

    MetaMem: Evolving Meta-Memory for Knowledge Utilization through Self-Reflective Symbolic Optimization

    Haidong Xin$^{*}$, Xinze Li$^{*}$, Zhenghao Liu$^†$, Yukun Yan$^†$, Shuo Wang, Cheng Yang, Yu Gu, Ge Yu, Maosong Sun

    MetaMem 借助自我反思式的符号优化让“元记忆”不断演化，从而在长程人机交互中更充分地利用知识。

    KDD 2026

    LISRec: Modeling User Preferences with Learned Item Shortcuts for Sequential Recommendation

    Haidong Xin, Zhenghao Liu$^†$, Sen Mei, Yukun Yan, Shi Yu, Shuo Wang, Zulong Chen, Yu Gu, Ge Yu, Chenyan Xiong

    LISRec 从用户的历史交互中学习个性化的语义捷径，更稳定地刻画用户偏好，进而提升序列推荐的效果。

    AI Open

    Knowledge Intensive Agents

    Zhenghao Liu, Pengcheng Huang, Zhipeng Xu, Xinze Li, Shuliang Liu, Chunyi Peng, Haidong Xin, Yukun Yan$^†$, Shuo Wang, Xu Han, Zhiyuan Liu$^†$, Maosong Sun$^†$, Yu Gu, Ge Yu

    我们从智能体视角系统梳理了检索增强生成（RAG），将知识密集型智能体划分为“知识获取”与“知识利用”两类，并探讨了多智能体 RAG 联合优化的未来方向。

    WISA 2025

    Adapting Language Models to Text Matching based Recommendation Systems

    Haidong Xin, Sen Mei, Zhenghao Liu$^†$, Xiaohua Li, Minghe Yu, Yu Gu, Ge Yu

    我们将语言模型的预训练能力引入推荐系统，显著提升了文本匹配任务的表现。

    EMNLP 2025

    LLMsPark: A Benchmark for Evaluating Large Language Models in Strategic Gaming Contexts

    Junhao Chen, Jingbo Sun, Xiang Li, Haidong Xin, Yuhao Xue, Yibin Xu, Hao Zhao$^†$

    LLMsPark 是一个面向策略博弈场景的大语言模型评测基准，可在多智能体环境中系统衡量模型的策略行为与决策能力。

    COLING 2024

    MMAD: Multi-modal Movie Audio Description

    Xiaojun Ye, Junhao Chen, Xiang Li, Haidong Xin, Chao Li, Sheng Zhou$^†$, Jiajun Bu

    MMAD 为视障人群带来了全新的观影体验，也拓展了电影音频描述在多模态方向上的能力边界。

    ArXiv 2026

    Memory Shot for Long-Term Dialogue

    Chunyi Peng$^{*}$, Haidong Xin$^{*}$, Xuanshuo Sheng, Xin Dai, Zhenghao Liu$^†$, Shuo Wang, Yukun Yan, Zulong Chen, Yu Gu, Ge Yu

    MemShot 将对话片段渲染为结构化的视觉记忆单元，借助模型自身的视觉推理关联关键片段，使长程对话的记忆构建提速达 70 倍。

    ICIC 2026

    CAGMamba: Context-Aware Gated Cross-Modal Mamba Network for Multimodal Sentiment Analysis

    Minghai Jiao, Jing Xiao$^†$, Peng Xiao, Ende Zhang, Shuang Kan, Wenyan Jiang, Jinyao Li, Yixian Liu, Haidong Xin

    CAGMamba 提出一种上下文感知的门控 Mamba 框架，用于多模态情感分析；它能刻画对话级别的情感演化，并在跨模态融合与单模态信息之间自适应地取得平衡。

    ArXiv 2026

    SEEK: Steering LLM Reasoning for RAG via Internal Reasoning Sketches

    Xinze Li,Yuqing Lan, Zhenghao Liu$^†$, Haidong Xin, Yukun Yan$^†$, Shuo Wang, Zheni Zeng, Sen Mei, Ge Yu, Maosong Sun

    SEEK 通过构建结构化的“认知提纲”来引导迭代检索，帮助 RAG 系统更有条理地组织多维知识，从而生成更准确的答案。

    MCM/ICM 2023

    Puzzle Game: Prediction and Classification of Wordle Solution Words

    Haidong Xin$^{*†}$, Fang Wu$^{*}$, Zhitong Zhou$^{*}$

    我们对 Wordle 游戏做了细致的数值分析，揭示了其中蕴含的统计规律。
