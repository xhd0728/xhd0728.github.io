    🔨 Projects

    GitHub Repo

    StaffDeck: Enterprise Digital Employee Platform

    ModelBest, NEU-ModelBest Data Intelligence Joint Lab, THUNLP, OpenBMB, and AI9Stars

    StaffDeck is an enterprise platform for building and managing digital employees. It turns professional experience, business processes, and decision criteria into digital employees that can work continuously, take over repetitive tasks, and preserve individual expertise as reusable, evolvable, and traceable organizational assets.

    GitHub Repo

    PilotDeck: A Workspace-Centric Open-Source Agent Operating System

    THUNLP, ModelBest, OpenBMB, and AI9Stars

    PilotDeck is an open-source agent operating system built around workspace-level isolation, white-box memory, smart routing, and always-on background execution for long-running productivity tasks.

    GitHub Repo

    UltraRAG 3.0: No More “Blind-Box” Development — Every Line of Reasoning Made Explicit

    THUNLP, NEU-ModelBest Data Intelligence Joint Lab, OpenBMB, and AI9Stars

    UltraRAG is a low-code MCP-based RAG framework that standardizes retrieval and generation components as MCP servers, with visual orchestration, evaluation, and demo deployment workflows.

    ACL 2026

    MetaMem: Evolving Meta-Memory for Knowledge Utilization through Self-Reflective Symbolic Optimization

    Haidong Xin$^{*}$, Xinze Li$^{*}$, Zhenghao Liu$^†$, Yukun Yan$^†$, Shuo Wang, Cheng Yang, Yu Gu, Ge Yu, Maosong Sun

    MetaMem is a meta-memory framework that keeps its memory evolving through self-reflective symbolic optimization, enabling more effective knowledge use across long-horizon human-LLM interactions.

    KDD 2026

    LISRec: Modeling User Preferences with Learned Item Shortcuts for Sequential Recommendation

    Haidong Xin, Zhenghao Liu$^†$, Sen Mei, Yukun Yan, Shi Yu, Shuo Wang, Zulong Chen, Yu Gu, Ge Yu, Chenyan Xiong

    LISRec strengthens sequential recommendation by learning personalized semantic shortcuts from user-item interaction histories, capturing stable user preferences more reliably.

    AI Open

    Knowledge Intensive Agents

    Zhenghao Liu, Pengcheng Huang, Zhipeng Xu, Xinze Li, Shuliang Liu, Chunyi Peng, Haidong Xin, Yukun Yan$^†$, Shuo Wang, Xu Han, Zhiyuan Liu$^†$, Maosong Sun$^†$, Yu Gu, Ge Yu

    A comprehensive survey of Retrieval-Augmented Generation from an agentic perspective: it organizes knowledge-intensive agents into knowledge-acquisition and knowledge-utilization roles, and outlines future directions for jointly optimizing multi-agent RAG systems.

    EMNLP 2025

    Junhao Chen, Jingbo Sun, Xiang Li, Haidong Xin, Yuhao Xue, Yibin Xu, Hao Zhao$^†$

    LLMsPark is a game-theoretic benchmark for evaluating the strategic behavior and decision-making of large language models in multi-agent environments, offering a new lens on model intelligence.

    COLING 2024

    MMAD: Multi-modal Movie Audio Description

    Xiaojun Ye, Junhao Chen, Xiang Li, Haidong Xin, Chao Li, Sheng Zhou$^†$, Jiajun Bu

    ArXiv 2026

    Memory Shot for Long-Term Dialogue

    Chunyi Peng$^{*}$, Haidong Xin$^{*}$, Xuanshuo Sheng, Xin Dai, Zhenghao Liu$^†$, Shuo Wang, Yukun Yan, Zulong Chen, Yu Gu, Ge Yu

    MemShot renders dialogue spans into structured visual memory units and uses the model’s visual reasoning to link key episodes, making long-term dialogue memory construction up to 70x faster.

    ICIC 2026

    CAGMamba: Context-Aware Gated Cross-Modal Mamba Network for Multimodal Sentiment Analysis

    Minghai Jiao, Jing Xiao$^†$, Peng Xiao, Ende Zhang, Shuang Kan, Wenyan Jiang, Jinyao Li, Yixian Liu, Haidong Xin

    CAGMamba is a context-aware gated cross-modal Mamba network for multimodal sentiment analysis that models dialogue-level sentiment evolution and adaptively balances cross-modal fusion against modality-specific cues.

    ArXiv 2026

    Xinze Li,Yuqing Lan, Zhenghao Liu$^†$, Haidong Xin, Yukun Yan$^†$, Shuo Wang, Zheni Zeng, Sen Mei, Ge Yu, Maosong Sun
